## 🎯 Project Overview
This interactive Excel dashboard was designed to analyze 1000+ rows of raw sales data to uncover regional performance trends.

## 🛠️ Key Features & Tools Used
* **Data Cleaning & ETL:** Handled missing values, removed duplicates, and structured data.
* **Data Modeling:** Created relational links between data tables.
* **Advanced Analytics:** Implemented `XLOOKUP`, `INDEX/MATCH`, and nested `IF` statements for dynamic metrics.
* **Interactivity:** Built dynamic charts integrated with **Slicers** for seamless user filtering.
